# 站外品牌提及 / 外链 / 平台分发执行清单

## 目标

把站外动作从“随便发一点内容”变成一套可持续执行的品牌提及和自然外链策略。

核心原则：

- 先做品牌提及，再做外链增长
- 先做高质量少量分发，再做扩大覆盖
- 优先把流量和信任导向服务页与文章页，不要只导首页
- 尽量用自然语言和品牌词，不做机械关键词锚文本

### 4. 知乎

适合发什么：

- 网站开发多少钱
- 企业官网需要哪些页面
- 外贸网站怎么提升询盘

执行方式：

- 先写答案，再整理成站内文章或反过来裁剪
- 回链尽量自然，不要生硬塞 3-4 个链接

### 5. 掘金

适合发什么：

- 偏产品和项目方法论
- 网站结构、SEO 基础、需求梳理、系统重构判断

执行方式：

- 从站内文章里挑更偏专业判断的内容同步
- 文末自然引到对应文章或服务页

### 6. 即刻

适合发什么：

- 短内容
- 项目判断
- 选型经验
- 接单边界

执行方式：

- 不必写长文
- 用短内容建立个人品牌和真实感
- 个人主页放官网链接

## 自然外链来源

优先争取：

- 老客户官网的友情链接或合作说明
- 朋友公司站点的资源页推荐
- 你自己的 GitHub 开源项目 README
- 你发在知乎 / 掘金 / Medium 的高质量内容
- 采访、播客、社区分享后的引用链接

不要做：

- 批量买垃圾外链
- 低质量站群
- 目录站批量提交
- 自动化评论区发链
