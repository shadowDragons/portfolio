ZenMind 冥想应用

集成 AI 生成音频技术的冥想辅助工具，极简主义设计风格

项目概述

ZenMind 是一款致力于帮助用户实现深度放松和专注力的冥想应用。项目核心挑战在于如何将 AI 生成的个性化音频与极简的视觉体验无缝融合。前端采用 Next.js 构建，利用其服务端渲染优势保证首屏加载速度；AI 音频生成引擎可根据用户的情绪状态和偏好动态生成自然环境音和引导语音。结合 Supabase 实现用户数据持久化，并通过 Stripe 完成订阅支付。最终的产品以其简约优雅的设计和智能化的体验获得了用户广泛好评。

技术栈

Next.js
AI/ML
Tailwind CSS
Supabase
Stripe

核心功能

AI 生成冥想音频

基于用户情绪状态与偏好，智能生成自然环境音、引导语音和背景氛围音乐。

专注力追踪

记录每次冥想时长与质量，生成可视化专注力曲线，帮助用户了解自身进步。

个性化推荐

根据用户的使用习惯和反馈，智能推荐最适合的冥想课程与音频组合。
