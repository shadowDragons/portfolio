KeyCustom 客制化商店

全 3D 的机械键盘定制商城，用户可实时预览不同轴体与键帽组合

项目概述

KeyCustom 是一个面向机械键盘爱好者的全 3D 客制化电商平台。用户可以在浏览器中通过 Three.js 实时预览键盘组装效果，从外壳、PCB 到轴体和键帽的每一个组件都可以自由选择和搭配。前端采用 Vue 3 配合 Pinia 状态管理，确保复杂的组件配置数据流转清晰；后端使用 Express 和 MongoDB 提供商品管理和订单服务。通过 3D 实时渲染技术，用户在确认购买前即可从任意角度查看自己的定制作品，极大提升了购物体验和转化率。

技术栈

Vue 3
Three.js
Pinia
Tailwind CSS
Express
MongoDB

核心功能

3D 实时预览

基于 Three.js 的高精度 3D 渲染，支持 360° 旋转查看，直观感受键盘每一个细节。

轴体组合系统

支持数十种轴体、键帽、外壳的自由搭配，实时更新价格与视觉效果。

一键分享配置

生成专属配置链接，方便用户在社区分享自己的键盘设计方案。
