CodeSync 协同编辑器

支持多人实时在线协作的轻量级代码编辑器，内置实时预览功能

项目概述

CodeSync 是一款面向开发团队的轻量级实时协同代码编辑器。项目旨在解决远程协作开发中的代码同步和实时预览难题。前端基于 React 和 CodeMirror 构建了高性能的代码编辑界面，通过 Socket.IO 实现毫秒级的操作同步，WebRTC 技术支持点对点的低延迟通信。内置的终端预览功能让团队成员无需切换窗口即可查看代码运行结果。版本历史回溯系统基于操作转换（OT）算法实现，确保多人同时编辑时不会产生冲突。项目采用 Docker 容器化部署，开箱即用。

技术栈

TypeScript
Socket.IO
React
CodeMirror
WebRTC
Docker

核心功能

实时协同编辑

支持多人同时编辑同一文件，实时同步光标位置和编辑内容，零冲突体验。

内置终端预览

集成 Web 终端，支持代码运行和即时预览，无需离开编辑器即可验证结果。

版本历史回溯

基于 OT 算法的版本控制系统，支持任意时间点的代码回滚与变更对比。
